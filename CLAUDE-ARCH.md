# Target Technologies

Framework: SvelteKit
TypeScript (strict mode)

## Frontend

- Svelte (components + pages)
- Tailwind CSS (styling)

## Infrastructure

- D1 via Drizzle ORM (database)
- R2 (file storage)
- Workers KV (sessions, config, cache)
- Zod (validation)
- REST API conventions

## Authentication

Firebase Auth with firebase-auth-cloudflare-workers

## Tooling

- Biome (formatting + linting)
- Vitest with @cloudflare/vitest-pool-workers (testing)
- Wrangler (dev server, types, deployment)
